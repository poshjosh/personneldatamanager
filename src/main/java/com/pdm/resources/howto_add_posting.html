How to Add Posting/Unit/Appointment

Home

While viewing personnel data table, do the following:

  • Right click on the first column of the row containing the data of the personnel for whom you want to add posting/unit/appointment, and a menu will pop-up.

    Indicating right click on table row
  • Select Add posting/unit/appointment from the pop-up Menu, and a form will be displayed.
  • Fill and submit the form.
