How to View Summary Report

Home

A Summary Report is created using 2 axis (X and Y axis). The X-axis is plotted against the Y-axis.

  • Click on the File in the Menu Bar, and a menu will appear.

    Indicating the Menu Bar
  • From the File Menu click on View Summary Report, and a pop-up Menu will appear for you to select axis options.

    Indicating summary report axis options
  • Select the type of each axis by clicking on the Combo Box for that axis. If you are not sure what to select, then select Rank for X-axis and Unit for Y-axis, then click OK.
